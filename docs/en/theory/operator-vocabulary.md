# Operator Vocabulary

Operators are typed transformation primitives.

They define kinds of change, not survival judgments.

## Rule

Transformation operators describe change.
Persistence theory evaluates what survives change.

| Code | Name      | Family        | Summary                                                    |
| ---- | --------- | ------------- | ---------------------------------------------------------- |
| CP   | copy      | structural    | Duplicate representation without terminating the source.   |
| PR   | project   | structural    | Expose a selected view while omitting other structure.     |
| EM   | embed     | structural    | Place structure inside a larger structure.                 |
| RF   | reference | structural    | Create an indirect pointer or citation.                    |
| RO   | reorder   | structural    | Rearrange declared parts.                                  |
| SP   | split     | decomposition | Separate one structure into multiple derived structures.   |
| MG   | merge     | aggregation   | Combine multiple structures into one derived structure.    |
| CL   | collapse  | compression   | Compress structure into reduced representation.            |
| EX   | expand    | elaboration   | Expose or generate additional internal structure.          |
| SH   | shift     | relocation    | Move structure across time, location, context, or frame.   |
| VS   | version   | evolution     | Create a controlled successor state with declared lineage. |
| BR   | branch    | evolution     | Create divergent continuities from a source.               |
| RV   | revert    | evolution     | Return toward a prior declared state.                      |
| BD   | bind      | contextual    | Attach a constraint, context, or dependency.               |
| UB   | unbind    | contextual    | Remove a constraint, context, or dependency.               |
| AZ   | authorize | normative     | Declare that a transformation pathway is permitted.        |
| AT   | attest    | observational | Assert evidentiary standing.                               |
